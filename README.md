
# 📚 Library Management System (C++)

## 📖 Description
A C++ OOP-based Library Management System to manage books and members, supporting book addition, issue/return, and search through a menu-driven interface.

## 🚀 Features
- Add Books
- Add Members
- Issue Books
- Return Books
- Search Books (Title/Author)
- Display Book List

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL (vector)

## ▶️ How to Run

1. Compile the program:
   g++ main.cpp -o library

2. Run the program:
   ./library
